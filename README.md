## 🎯 Business Objective
Analyze Airbnb listing data to identify pricing trends, host performance, property distribution, and location-based insights.

## 🛠 Tools & Technologies
- Python, Pandas, Jupyter Notebook  
- Power BI  
- GitHub  

## 🧹 Data Cleaning
- Removed duplicates & null values  
- Formatted price into numeric/currency  
- Created price categories  
- Exported cleaned CSV for dashboarding  

## 📊 Dashboard Highlights
- Built interactive Power BI dashboard with slicers and filters  
- Designed KPI cards and analytical visuals  
- Structured insights for business understanding  

## 📌 KPI Metrics
- Total Listings  
- Average Price

## 📈 Visual Analysis

- Location-based Listing Insights
